# Remaining risks

| Risk | Mitigation |
|------|------------|
| OpenRouter BAA | Use `AI_PHI_MODE=restricted` or switch to BAA-covered provider |
| Platform admin power | MFA, narrow allowlist, audit impersonation |
| Single-region deployment | Multi-region when customer base requires |
| Email deliverability | Configure Resend/SendGrid for invites and intake links |
