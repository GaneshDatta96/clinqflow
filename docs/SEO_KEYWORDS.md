## Search intent guide

- **Informational** — User researching how intake or documentation workflows work. Target with FAQ, how-it-works pages, and homepage workflow section.
- **Commercial** — User comparing clinic software. Target with landing pages and homepage product section; CTA is always **Sign up** → `/signup`.
- **Transactional** — User ready to create an account. Homepage CTA bands and landing page CTAs; do not index `/signup` (noindex).

## Usage rules

1. One primary keyword cluster per landing page; support with natural H2/H3 variants.
2. No keyword stuffing; prioritize clarity and compliance over density.
3. Do not target competitor brand names or “vs” comparison pages in this phase.
4. Use “AI-assisted documentation” and “clinical decision-support for practitioner review” — not “AI diagnosis” or “automated clinical decisions.”
